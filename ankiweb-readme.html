⚠️ DEPRECATION NOTICE
This add-on version is deprecated and will no longer receive feature updates. It has been replaced by the v2 version: Greg’s Code Highlighter.
An Anki plugin that creates highlighted code blocks. This plugin uses a colorscheme that works in day and night modes. It also works on both, Anki for desktop and AnkiDroid. Migration to v2
  1. Install the new v2 add-on using code: 1527277801 (and restart Anki).
  2. In Anki’s menu bar, run Tools > Migrate Code to v2 Pygments.
  3. Uninstall this v1 add-on, and restart Anki.
Caveats
This add-on adds a custom stanza to your card templates. This is necessary to have the add-on work in an upgradeable manner. The add-on provides a convenient menu command to removing all stanzas if you decide you want to remove the plugin. Use
  1. Select text that you want to create a code block from or copy code to clipboard and put a cursor at its desired destination.
  2. Press CTRL+' or click the add-on’s button.
