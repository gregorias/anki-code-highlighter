An Anki plugin that creates highlighted code blocks. This plugin uses a colorscheme that works in day and night modes. It also works on both, Anki for desktop and AnkiDroid. Caveats
This add-on installs its own CSS files into Anki’s media folder. If you decide to fully delete the add-on and wish to remove these files, please follow the uninstall instructions in the README. Use
  1. Select text that you want to create a code block from or copy code to clipboard and put a cursor at its desired destination.
  2. Press CTRL+o or click the add-on’s button.
For more information, consult the README. Feedback If you have any suggestions, let me know at this plugin’s GitHub. I use the plugin personally, and I’m open to improving it.
